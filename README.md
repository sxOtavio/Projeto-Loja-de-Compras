Sistema de Produtos com Login (Node.js + Express)
Este é um projeto web desenvolvido em Node.js, utilizando Express, MySQL, EJS e Sessions, com foco em autenticação de usuário, controle de acesso e CRUD de produtos com upload de imagens.
Funcionalidades

      Sistema de login com sessão
      Rotas protegidas (acesso somente após login)
      Cadastro de produtos
      Exclusão de produtos
      Upload e remoção de imagens
      Listagem de produtos via banco de dados
      Integração com MySQL
      Templates com EJS
Tecnologias Utilizadas

    Node.js
    Express
    MySQL
    EJS
    express-session
    express-fileupload
    fs (File System)
    path
